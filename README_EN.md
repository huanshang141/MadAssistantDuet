> MAD stands for MAD Assistant Duet

## how to use
Unfortunately, this program currently only supports Simplified Chinese text because it uses OCR for recognition. You can adjust the language settings in the game to use it.

1. you must set Setting-Display-Display mode to Borderless Windowed, and set Setting-Language-Text language to 简体中文.

2. right-click `MFAAvalonia.exe` and select "Run as administrator"

3. you can run the "Start Task" in the script once you have entered the stage.

## task
And currently, it only supports specific stages from "Noctoyager Manual", depending on which options are selected.

"闪避键设置" means setting the key of dodge, default as L Shift

"自动战斗设置" means strategy of auto battle when arrive at destination, "循环E" means pressing E Periodically, "无操作" means doing nothing

"65级mod（扼守）" means all the stages in Noctoyager Manual lv65, but only lv65
